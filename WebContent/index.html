About ChEA3 [still working on this...]

Compute enrichment of your gene set with libraries of putative transcription factor targets.

  • Transcription Factor Target Enrichment Fisher's Exact Tests are used to compare the input gene set with putative transcriptional target sets for each human transcription factor. The census of human transcription factors described in xx was used to define the set of transcription factors used in this tool. Putative transcription factor target sets are organized into "libraries" which generally correspond to Try it now!
  • Transcription Factor Target Libraries The transcription factor target libraries were compiled from multiple sources (Figure 2). Genes highly co-expressed with transcription factors were pulled from GTEx2 and uniformly reprocessed GEO RNA-seq experiments via ARCHS43. Uniformly reprocessed public ChIP-seq data were used from ENCODE4 and ReMap5. Putative targets were compiled with a high TF-gene co-occurrence in crowd-submitted queries to the tool Enrichr6. ChIP-seq data was manually mined from the literature1. Gene expression signatures from single-TF loss-of-function (LOF) and gain-of-function experiments (GOF) mined from GEO were used as a benchmarking dataset8.
  • Benchmarking Predictions Pairwise Fisherâ€™s Exact Tests (FETs) were conducted between the gene sets of the TF LOF-GOF library and the gene sets of all the other libraries with the goal of correctly predicting the TF that was perturbed in each of the GOF/LOF library experiments. For each LOF/GOF experiment, predicted TFs were ranked according to the significance of the FETs. Rank aggregation was performed among the ranked results of all of the libraries to yield a list of integrated results. Integrating the results improves the performance of this approach (Figure 3).

ChEA3 FAQ

Which library should I use?

Ultimately this will depend on your application, however in our benchmarking studies, the "Integrated" library performed the best. The Integrated library uses a rank aggregation scheme to integrate results from all other ChEA3 libraries.

Why ChEA3?

Key differences between ChEA3 and other TFEA tools are that ChEA3 uses libraries of putative transcription factor targets compiled from multiple difference sources and assays.

What can I use ChEA3 for?

ChEA3 will rank transcription factosr according to their predicted transcriptional influence over your gene set. For case studies, please see the ChEA3 paper.
