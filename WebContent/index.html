About ChEA3

Transcription factors (TFs) are proteins that control gene expression by binding and unbinding near coding regions to regulate the transcriptional machinery. TF enrichment analysis (TFEA) prioritizes transcription factors based on the overlap between given lists of differentially expressed genes, and previously annotated TF targets assembled from multiple resources. ChEA3 is a web-based TFEA tool. Hence, ChEA3 can aid in identifying the TFs responsible for observed changes in gene expression when comparing control and perturbation samples. In the past, we have developed and published ChEA and ChEA2 which are ChIP-seq enrichment analysis tools made of gene set libraries created from published ChIP-seq data extracted from multiple sources. ChEA3 builds upon these prior versions of ChEA by including more libraries, adding benchmarks, and integrative library analyses.

For ChEA3 the following TF-target gene set libraries were assembled: Putative targets as determined by ChIP-seq experiments from ENCODE, ReMap, and individual publications; co-expression of TFs with other genes based on processed RNA-seq from GTEx and ARCHS4; co-occurrence of TFs with other genes by examining thousands of gene lists submitted to the tool Enrichr; and gene signatures resulting from single TF perturbations followed by genome-wide gene expression experiments.

The ChEA3 web application outputs enrichment results in the form of searchable, sortable data tables for each library and integration method. An interactive TF co-expression network visualization, clustered using weighted gene co-expression network analysis, places the top enrichment results in the context of the larger human transcriptional regulatory network. Using various benchmarks, we found that integrating libraries boosts performance of ranking TF correctly.

Transcription Factor Target Enrichment Analysis - The goal of ChEA3 is to predict transcription factors (TFs) associated with user-input sets of genes. Discrete query gene sets are compared to ChEA3 libraries of TF target gene sets assembled from multiple orthogonal 'omics' datasets. The Fisher's Exact Test is used to compare the input gene set to the TF target gene sets in order to determine which TFs may be most closely associated with the input gene set. The gene set background used to perform these tests is comprised of all approved HGNC gene symbols.

Input - Users can submit a set of human or mouse gene symbols for transcription factor enrichment analysis. The input gene set can be derived from a variety of sources. Commonly, the gene set may be a set of differentially expressed genes, with the goal of forming hypotheses about which TFs might be responsible for the changes in gene expression. However, input from GWAS studies or whole-genome CRISPR-KO screen hits can be used as input to identify common regulators for these sets of genes.

Network Visualization - The ChEA3 application displays a scatter plot of points representing all human transcription factors based on their co-expression similarity. This TF coexpression network was constructed by conducting a Weighted Gene Co-expression Network Analysis (WGCNA) over RNA-seq counts of human TFs over all GTEx samples. The network layout was determined using Allegro Edge-Repulsive Strong Clustering in Cytoscape. WGCNA TF module summary expression profiles, also known as Eigen-genes, were correlated with gene expression profiles from all GTEx tissue samples to determine strong module-tissue associations. Each WGCNA module may be colored by its most highly correlated tissue. Tissues are aggregated into more general tissue types (e.g. brain) and more specific tissue types (e.g. cerebellum, frontal cortex).

Benchmarking Predictions - The performance of each library and integration method was benchmarked using 946 single-TF loss-of-function (LOF) and gain-of-function (GOF) experiments mined from GEO with microarray and RNA-seq readouts. Gene sets were generated from the differentially expressed genes of these experiments and submitted to the ChEA3 API. The positive class consists of ranks assigned by ChEA3 to perturbed TFs. The negative class consists of ranks assigned by ChEA3 to TFs not perturbed in the experiment. The areas under the receiver operating characteristic (ROC) and precision-recall (PR) curves were used to evaluate how well ChEA3 ranked the true perturbed TF. Each library has a different number of TFs covered. To enable comparison, the negative class was downsampled to the size of the positive class to generate PR curves such that a random classifier has an AUC of 0.5. These PR curves were bootstrapped 5,000 times to generate mean PR AUCs and composite PR curves.

Tutorial

Discover transcription factors associated with your gene set

  1. Submit Your Query - Submit a query with one gene symbol per row. ChEA3 accepts HGNC-approved gene symbols and will discard probe names, transcript IDs, and other unrecognizable IDs that have not been converted to gene symbols. You can use the example gene set available as a link above the submission box to gain familiarity with the expected format. Once submitting your query, you will be presented with the results page. Most gene set queries are processed in less than 30 seconds.
  2. Navigating Your Results - Results appear as collapsible tables in the results side pane. Clicking on the table header expands the table. Tables may be searched by entering terms in the search box, sorted by column by clicking on the column header, and copied to the clipboard by clicking on the Copy button. Results for the ChEA3 TF target libraries are sorted by the Fisher's Exact Test p-value. This provides a ranking of transcription factors whose putative transcriptional targets are most closely similar to the query set. Integrated results, which take into account results from all libraries, are sorted in ascending order by score. Lower scores indicate more relevancy to the transcription factor. The top 100 results are listed in each table. The full results may be downloaded in JSON format using the button at the bottom of the results pane. Full results are also available via the ChEA3 API.
  3. Interpreting Your Results - The ChEA3 transcription factor rankings have been evaluated by benchmarking the application using query gene sets generated from single-transcription factor perturbation followed by genome-wide expression experiments. The performance of ChEA3 in highly ranking the "true" perturbed transcription factor is summarized using Receiver Operating Characteristic (ROC) and Precision-Recall (PR) curves. The area under the curves for each TF library, as well as two methods that integrate library results (Borda count and a Top Rank method) are summarized above (Table 1). For convenience, the results from the benchmarking exercise are used to rank the libraries in the results pane. Hence, we recommend that users consider the quality of their enrichment results in each results table based on this order which represents overall quality rank.
  4. Visualizing Your Results - A transcription factor co-expression network is provided to help users visualize their top matching transcription factors in the context of the larger human transcriptional regulatory network. Network coloring options provide additional information about the tissues the transcription factors may be most active. Sliders and color pickers on each table enable users to highlight the number and color of the matching TFs. The network may be zoomed, panned, and saved as an SVG image. There are other options, for example, hiding the legend, switching to full screen, or showing/hiding node labels. These options can be accessed by clicking the hamburger near the network title.

API

ChEA3 has an API to enable programmatic access for querying the ChEA3 database

The ChEA3 REST API uses GET requests to transport user submitted gene sets and JSON-formatted query results between the ChEA3 server and the user’s script. The user gene set and optional additional information are encoded in the GET URL. To query ChEA3, the GET URL is constructed by appending to the base URL: 
http://amp.pharm.mssm.edu/chea3/api/enrich/
For example: 
https://amp.pharm.mssm.edu/chea3/api/enrich/FOXO1,MYC,EGR1,RELB,ICAM1,ARIDB4
  1. Command-line Example - The following returns results from all ChEA3 libraries in JSON format:
    $ curl -H "Accept: application/json" https://amp.pharm.mssm.edu/chea3/api/enrich/FOXO1,MYC,EGR1,RELB,ICAM1,ARIDB4
    Redirect the JSON-formatted results to a file: 
    $ curl -H "Accept: application/json" https://amp.pharm.mssm.edu/chea3/api/enrich/FOXO1,MYC,EGR1,RELB,ICAM1,ARIDB4 > results.json
  2. R Example - 
    								library(jsonlite)
								library(httr)
								base_url <- "https://amp.pharm.mssm.edu/chea3/api/enrich/"
								gene_set <- c("FOXO1","MYC","EGR1","RELB","ICAM1","ARIDB4")
								#collapse gene set to 'FOXO1,MYC,EGR1,RELB,ICAM1,ARIDB4'
								gene_set <- paste(gene_set,collapse=",")
								call = paste(base_url,gene_set,sep='')
								response = GET(call)
								json = content(response,"text")
								#results as R list of dataframes
								results = fromJSON(json)

Download

Libraries used for ChEA3 enrichment analysis

ARCHS4 Coexpression

ARCHS4 is a massive RNA-seq reprocessing effort containing over 150,000 human and mouse samples from GEO. The ARCHS4 TF co-expression library was generated by randomly sampling 50,000 human RNA-seq samples from ARCHS4, building a gene-gene Pearson correlation matrix, and then taking the 300 genes with the greatest absolute value of the Pearson correlation coefficient with each of 1,618 unique human transcription factors for a total of 1,618 gene sets. The library encompasses 485,400 putative TF-gene interactions.

GTEx Coexpression

The Genotype-Tissue Expression (GTEx) project is a public resource dedicated to cataloguing tissue-specific gene expression and regulation. Samples were collected from 53 healthy tissue sites across almost 1000 individuals, for molecular assays including whole-genome sequencing, whole-exome sequencing, and RNA-seq. All human GTEx RNA-seq samples were used to build a gene-gene Pearson correlation matrix across all tissues available. The 300 genes with the greatest absolute value of the Pearson correlation coefficient for each of 1,598 unique human transcription were used to build the TF gene set library. The library encompasses 479,400 putative TF-gene interactions.

Enrichr Queries

User-submitted gene lists were copied from the Enrichr for a total of 1,097,157 unique lists. Enrichr example lists, lists used for internal testing, lists with greater than 2000 genes, lists with less than 2 genes, and lists from IP addresses that submitted >1000 lists were filtered out which resulted in 293,747 unique gene lists remaining for co-occurrence analysis. For each TF i defined in a census of 1404 TFs, the probability of co-occurrence of the transcription factor TFi and a gene gj P(TFi ∩ gj) was computed for all genes in G. This is essentially the count of the number of times TFi and gj appear together in the same list across the user-submitted gene lists. The top 300 co-occurring genes for each TFi were used. to generate the GMT gene lists. The library encompasses 419,114 TF-gene interactions.

ReMap ChIP-seq

The ReMap ChIP-seq library was generated using BED files from the ReMap Project. BED files were annotated with a distance-weighted peak-to-gene model and the top 10% of transcription start sites were assigned to each of 300 unique TFs to generate putative target sets, encompassing 422,025 TF-gene interactions.

ENCODE ChIP-seq

TF target gene sets were generated for the web tool Enrichr using uniformly reprocessed ENCODE TF ChIP-seq experiments. Peak calling on alignment files was conducted using the MACs software. Peaks were then sorted by distance to the transcription start site and 2000 peaks at most were retained for each experiment. The library encompasses 552 ChIP-seq experiments of 118 unique TFs. The library contains 867,121 TF-gene interactions.

Literature ChIP-seq (ChEA)

This library is derived from TF ChIP-seq and ChIP-ChIP experiments mined from supplemental data in the literature. Previous versions of this library were used in ChEA and ChEA2. The library gene set metadata includes the PubMed ID number for the publication from which the experiment originated, as well as the species, assay and the cell- or tissue-type. This library encompasses 463 ChIP-seq and ChIP-ChIP experiments across 282 publications. The gene sets are associated with 194 unique TFs and contains 623,233 TF-gene interactions.

Benchmarking Libraries

TF LOF/GOF Experiments

Single-TF perturbation experiments including knockdowns, knockouts, overexpression, and chemical inhibition followed by genome-wide microarray profiling were manually curated by the crowd for the from CREEDS. An additional 160 single-TF perturbation from RNA-seq experiments were manually curated from GEO by first programmatically searching ARCHS4 metadata for potential studies that contain relevant signatures. The top 300 upregulated, and the top 300 downregulated genes were used to compose TF perturbation experiment - associated gene sets for a total of 786 gene sets associated with 275 unique site-specific TFs.

TF shRNA Experiments in B-LCLs

The top 300 upregulated and top 300 downregulated genes from each of 49 unique TF shRNA knockdowns by Cusanovich et al. were used to generate 49 TF perturbation-associated gene sets.

Additional Libraries

Hu.MAP TF PPIs

Hu.MAP integrates large-scale affinity purification mass-spec datasets. All putative protein interactors with each TF were used to generate this TF PPI library.

iRefWeb TF PPIs

iRefWeb integrates PPIs from several major public PPI databases. PPIs from the BioGRID, MPPI, CORUM, and HPRD databases are included in this library.
