About ChEA3

Transcription Factor Target Enrichment Analysis - The goal of ChEA3 is to predict transcription factors (TFs) associated with user-input sets of genes. Discrete query gene sets are compared to ChEA3 libraries of TF target gene sets assembled from multiple orthogonal 'omics' datasets. Fisher's Exact Tests are used to compare the input gene set with TF target gene sets in order to determine which TFs may be most closely associated with the input gene set. The gene background used to perform these tests is comprised of all approved HGNC gene symbols. The primary goal of ChEA3 is to establish a ranking of transcriptions factors that may be associated with the input gene set. The background composition has no consequence on this ranking, though it may affect the tests' statistical significance. Integrating results from multiple libraries provides better TF predictions than each library alone (see Benchmarking below).

Input - Users submit a discrete set of gene symbols for transcription factor target enrichment analysis. This gene set can be derived from a variety of sources. Commonly, the gene set may be a set of differentially expressed genes, with the goal of forming hypotheses about which TFs might be responsible for the changes in gene expression.

Network Visualization - The TF Co-expression Network was constructed by conducting a Weighted Gene Co-expression Network Analysis (WGCNA) over RNA-seq counts of human TFs over all GTEx samples. The network layout was determined using Allegro Strong Clustering in Cytoscape. WGCNA TF modules are colored by the tissue(s) that the module eigengene is most closely correlated with.

Benchmarking Predictions - The performance of each library and integration method was benchmarked using 979 single-TF loss-of-function (LOF) and gain-of-function (GOF) experiments mined from GEO with microarray and RNA-seq readouts. Gene sets were generated from the differentially expressed genes of these experiments and the areas under the receiver operating characteristic (ROC) and precision-recall (PR) curves were used to evaluate how well ChEA3 ranked the true perturbed TF.

API

ChEA3 provides a simple API to allow programmatic querying with your gene sets

The ChEA3 REST API uses GET requests to transport your gene set and JSON-formatted query results between the ChEA3 server and your scripts. Your gene set and optional additional information is encoded in the GET url. To query ChEA3, the GET url is constructed by appending the url 
http://amp.pharm.mssm.edu/chea3/api/enrich/
to your comma-delimited gene set. For example: 
https://amp.pharm.mssm.edu/chea3/api/enrich/FOXO1,MYC,EGR1,RELB,ICAM1,ARIDB4
  1. Command-line Example - 
    $ curl -H "Accept: application/json" https://amp.pharm.mssm.edu/chea3/api/enrich/FOXO1,MYC,EGR1,RELB,ICAM1,ARIDB4
    Redirect the json-formatted results to a file: 
    $ curl -H "Accept: application/json" https://amp.pharm.mssm.edu/chea3/api/enrich/FOXO1,MYC,EGR1,RELB,ICAM1,ARIDB4 > results.json

Download Libraries

ARCHS4 Coexpression

ARCHS4 is a massive RNA-seq reprocessing effort containing over 150,000 human and mouse samples from GEO. The ARCHS4 TF co-expression library was generated by randomly sampling 50,000 human RNA-seq samples from ARCHS4, building a gene-gene Pearson correlation matrix, and then taking the top 300 genes that are correlated/anti-correlated with each of 1,618 unique human transcription factors for a total of 1,618 gene sets. The library encompasses 485,400 putative TF-gene interactions.

GTEx Coexpression

GTEx is a public resource dedicated to cataloguing tissue-specific gene expression and regulation. All human GTEx RNA-seq samples were used to build a gene-gene Pearson correlation matrix across all tissues available. The top 300 genes that are correlated/anti-correlated with each of 1,598 unique human transcription were used to build the TF gene set library. The library encompasses 479,400 putative TF-gene interactions.

Enrichr Queries

User-submitted gene lists were copied from the Enrichr database and filtered, resulting in 293,747 unique gene lists. For each of 1,404 transcription factors, the probability of co-occurrence of the transcription factor and any given gene was computed. The top 300 co-occurring genes for each TF were used to generate the TF gene sets. The library encompasses 419,114 TF-gene interactions.

ReMap ChIP-seq

The ReMap ChIP-seq library was generated using bed files from the ReMap Project. Bed files were annotated with a distance-weighted peak-to-gene model and the top 10% of of transcription start sites were assigned to each of 300 unique TFs to generate putative target sets, encompassing 422,025 TF-gene interactions.

ENCODE ChIP-seq

TF target gene sets generated for the web tool Enrichr using uniformly reprocessed ENCODE TF ChIP-seq experiments.

Literature ChIP-seq (ChEA)

This library is derived from TF ChIP-seq experiments mined from supplemental data in the literature. Previous versions of this library were used in ChEA and ChEA2.
